HTML:

HTML stands for Hypertext Markup Language. All web pages are built on HTML and that's why it's important
to learn.

Tag and Elements:

HTML documents are made up of HTML elements. When writing HTML, we tell browsers the type of element by
using HTML tags. Here is an unrealted link to espn.

Computers are stupid:

There is a really cool way we explain why computers are stupid to kids. I heard this on a podcast once. Have a
teacher pretend to be a computer that has to make a PB and J sandwhich. The teacher takes literal orders from the kids and
makes a mess of things. The kids learn that telling a computer to make a sandwhich is a complex and detailed task.


Inline vs Block Elements:

HTML elements are either inline or block. The difference is that block elements have an "invisble box" around them
used to format the element.
